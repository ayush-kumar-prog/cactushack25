# PULSE

### AI-Powered Emergency Response Copilot

Anyone, anywhere on earth can now perform Basic Life Support, powered by our edge AI BLS copilot, **Pulse**. Pulse guides you through doing BLS and autonomously calls first responders with context.

---

## QUICK START

1. **Download** `Pulse.apk` from this repository
2. **Install** on any Android phone
3. **Open** the app and grant camera/microphone permissions
4. **Say** "Someone collapsed!" to begin

---

## For Judges

**Twilio Integration:** Autonomous emergency calls and SMS alerts are active during the live demo (backend running on presenter's laptop, same WiFi network). The core AI guidance works standalone on any device.

---

## Features

- Voice-activated AI emergency guidance
- Real-time patient assessment via camera
- AR overlays for CPR positioning
- Autonomous 911 call with patient context
- Family SMS alerts with location
- Fully hands-free operation
